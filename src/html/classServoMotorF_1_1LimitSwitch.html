def ServoMotorF.LimitSwitch.checkswitch (   self)

Checks the limitswitch value and determines if the duty needs to be set to zero.

The function checks the value of the switchpin1 and if the value is 0 then it will set the duty for the R motor to zero.

ServoMotorF.LimitSwitch.duty2

Shared Variable for duty of motor 2.

This shared variable for the duty of motor 2 is set up to be a float. This is important because when we write to this variable in our closed loop controller, the number we calculate is a float.

◆ Rswitch

ServoMotorF.LimitSwitch.Rswitch

Shared variable for the value of our R limit switch.

This variable contains the value of the pin that our limit switch is connected to. We are using this limit switch as a kill switch so that when the pin value changes from 1 to 0, the duty to all our motors is set to 0. This prevents our motors from burning up when the parts reach a mechanical stop.
