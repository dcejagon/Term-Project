Detailed Description

Interface with quadrature encoders.

This code allows us to calculate the current position in ticks of the motor shaft

def REncoder.EncoderDriver.read (   self)

Reads encoder 2 position.

This function is responsible for reading the encoder position and delta values, at a constant interval defined in the main program file. Instead of simply reading timer.counter(), we must calculate the delta of function so that our encoder does not overflow when it reaches the period (65535 ticks) This function does not 'return' anything, but it writes to our shared variable for encoder position.
