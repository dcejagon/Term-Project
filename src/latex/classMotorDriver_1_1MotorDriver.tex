This function allows us to set a duty cycle for the motor 1. 

This method sets the duty cycle to be sent to the motor at a given level. Positive values cause rotation in one direction, negative values in the opposite direction. We accomplish this rotation by using the pulse\+\_\+width\+\_\+percent(\#) method for each of our channels. By applying \# to channel 1 and 0 to channel 2, the motor will rotate one way. By applying 0 to channel 1 and \# to channel 2, the motor will rotate the opposite way.
