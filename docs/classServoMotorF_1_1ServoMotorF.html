def ServoMotorF.ServoMotorF.down (   self)

This function allows us to send a PWM signal to move servo down.

This method sends a PWM signal to the servo motor which is interpreted as a position, as the PWM % decreases the pen moves down.

Returns
Returns value, 2, associated with down position.
