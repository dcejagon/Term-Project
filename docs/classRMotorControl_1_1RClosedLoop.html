def RMotorControl.RClosedLoop.control_loop (   self)

Runs closed loop control on R motor.

This method runs the closed loop control system for the R motor. we accomplish this by multiplying the difference between the setpoint and the current position by a porportional gain. We then write this value to the duty that will be passed into our motor driver.
