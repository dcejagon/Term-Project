Detailed Description

Puts system into a closed loop.

This code allows us to run the motor/encoder system within a closed loop. by calculating the actuation signal from the difference between the current setpoint and current position multiplied by gain Kp1.

Runs closed loop control on theta motor.

This method runs the closed loop control system for the theta motor. we accomplish this by multiplying the difference between the setpoint and the current position by a porportional gain. We then write this value to the duty that will be passed into our motor driver.
