{\footnotesize\ttfamily Servo\+Motor\+F.\+Limit\+Switch.\+Rswitch}



Shared variable for the value of our R limit switch. 

This variable contains the value of the pin that our limit switch is connected to. We are using this limit switch as a kill switch so that when the pin value changes from 1 to 0, the duty to all our motors is set to 0. This prevents our motors from burning up when the parts reach a mechanical stop. \mbox{\Hypertarget{classServoMotorF_1_1LimitSwitch_a58244c655e8ad42abcee7d25a1628627}\label{classServoMotorF_1_1LimitSwitch_a58244c655e8ad42abcee7d25a1628627}}
